💥七月计划 |【基础扎实月】（目标：能写小型自动化脚本）
Week 1（7.1 - 7.7）| 环境准备＋语言复习
Python 基础语法（控制流、函数、模块、文件操作、异常处理）

安装 PyCharm / VSCode，配置虚拟环境

了解 Pytest 框架的基本用法

✅输出：写出一个基础自动化脚本（比如自动登录网页并截图）

Week 2（7.8 - 7.14）| Selenium 入门
安装 Selenium、驱动配置

元素定位（ID、Xpath、CSS 等）、页面操作（点击、输入、等待）

编写简单 UI 自动化脚本（如打开网页→输入→点击→断言）

✅输出：自动化“登录-搜索-退出”一套流程

Week 3（7.15 - 7.21）| Pytest + Selenium 实战
Pytest 的 fixture、断言、参数化

报告生成插件（Allure 简单上手）

编写模块化、结构化的测试用例

✅输出：完成一个小型测试项目，结构清晰、能跑、能报错、能出报告

Week 4（7.22 - 7.28）| CI/CD基础 + 数据驱动 + 提升
数据驱动测试（读取 Excel / YAML）

Git 基础操作

简单接触 CI 工具（如 GitHub Actions / Jenkins 简介）

✅输出：测试用例 + 数据文件 + 一次 push + 报告

🧨八月计划 |【进阶＋投递月】（目标：完成简历项目 & 投出简历）
Week 5（7.29 - 8.4）| 编写自己的简历项目
构建一个完整的项目目录结构

模拟实际业务流程（注册、登录、下单…）

配置 config 文件、报告、美化脚本结构

✅输出：一个可打包讲解的项目，能写进简历那种

Week 6（8.5 - 8.11）| 项目优化＋面试准备
熟练调试、找 bug、加点异常处理

开始整理面试问题（用法＋原理＋错误排查）

Mock 面试：你说我答，我来打磨你的表达

✅输出：简历 + GitHub 项目 + 面试 Q&A 初稿

Week 7 & 8（8.12 - 8.25）| 大规模投递＋实战应对
大量投递简历，优先广深，自动筛选“初级岗位”

每次面试完总结：技术盲区、问题优化

实时调整简历/技术细节，保持动态优化

✅输出：5-10个以上投递记录 + 3次以上面试经验
